🧰 أدوات المطوّر

مجموعة أدوات تشخيص لمشروع زاد — كل أداة تفحص جانباً مختلفاً من جودة الموقع.
🔍
فاحص الكونتراست
يفحص كل نصوص الموقع ويقيس كونتراست WCAG، يبرز الضعيف في كل الأوضاع (فاتح/داكن/أوليد/عذر).
←
📋
متتبّع المحتوى
يرصد placeholders والأقسام الفاضية والروابط المكسورة آلياً، + تتبّع يدوي للنواقص الشرعية.
←
🌍
محاكي المكان/الزمان
يحاكي GPS وأوقات مختلفة (مكة/أوسلو/ريكيافيك...) لاختبار حسابات الصلاة والقبلة + اختبار أوفلاين.
←
🔐
فاحص الأمان
يكشف ثغرات الأمان الشائعة: Mixed Content، SRI ناقص، iframes بلا sandbox، secrets، eval().
←
🎨
استوديو الثيمات والزخارف
صمّم ثيمات مخصّصة (ألوان + باترن إسلامي/glassmorphism) بمعاينة حيّة، وصدّر كود CSS جاهزاً للتطبيق.
←
⚡
فاحص الأداء
يقيس Core Web Vitals الحقيقية (LCP/CLS/FCP) وأزمنة التحميل والموارد والذاكرة من APIs المتصفح.
←
♿
فاحص إمكانية الوصول
يفحص معايير WCAG: alt الصور، ARIA، labels الحقول، ترتيب العناوين، التنقّل بالكيبورد.
←
💡 ملاحظة: هذه الأدوات للتطوير فقط. كل أداة لها زرّ «فحص كل الموقع» يفحص الصفحات تلقائياً، وزرّ «نسخ التقرير» لمشاركة النتائج. أغلبها يدعم bookmarklet تسحبه لشريط المفضّلة لفحص أي صفحة مباشرةً.
